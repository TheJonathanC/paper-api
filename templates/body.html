Exam Paper API

  This API has two "extensions".



1. Return all data

 expaper-api.onrender.com/exampapers


2. Return for specific subject

 expaper-api.onrender.com/exampapers?code=178 (BCA)

  • BCA - 178
  • BBA - 177
  • BSc - 176
  • BCom - 174
  • BA - 274

Click on the white icon to go to the respective "extension code"
